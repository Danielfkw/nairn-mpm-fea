The nairn-mpm-fea Project Source Code Organization

Assuming that checked out folder's name is nairn-mpm-fea, the NairnMPM, NairnFEA, Documentation, and Visualization source code is organized into various directories. The contents of the directories are:

nairn-mpm-fea/Common
This folder has items common to both MPM and FEA code
/Projects
This folder has XCode and Visual Studio project files for editing and compiling the code.
/Programming Notes
Development tasks and notes (in OSParticulas only).
/(other folder)
All other folders in Common have various common sections of code.
nairn-mpm-fea/NairnMPM
This folder is the root folder for the NairnMPM source code
/src
This folder has all MPM-specific source code in various subfolders
/build
This folder has the makefile for compiling the code using a make command.
/input
This folder has sample input files and the NairnMPM.dtd file. The compiled code gets installed here ready for running with the input files.
/Scripted_Input
This folder has scripted input files that can be run using with NairnFEAMPM or NairnFEAMPMViz applications.
/XML_Input
This folder has XML input files that can be run from a command line. It is recommended that you rarely use such files and instead do all calculations using the scripting tools in NairnFEAMPM and NairnFEAMPMViz.
/Test-Suite
A RunTests script for running many examples and comparing to previous code (in OSParticulas only).
/MMSValidations
Command files for using manufactured solutions to validate code development.
/tools
This folder has source code for command line tools that can be used with NairnMPM such as for extracting data from archive files or for comparing global results files.
nairn-mpm-fea/NairnFEA
This folder is the root folder for the NairnFEA source code
/src
This folder has all FEA-specific source code in various subfolders
/build
This folder has the makefile for compiling the code using a make command.
/input
This folder has sample input files and the NairnFEA.dtd file. The compiled code gets installed here ready for running with the input files.
/Scripted_Input
This folder has scripted input files that can be run using with NairnFEAMPM or NairnFEAMPMViz applications.
/XML_Input
This folder has XML input files that can be run from a command line. It is recommended that you rarely use such files and instead do all calculations using the scripting tools in NairnFEAMPM and NairnFEAMPMViz.
/Test-Suite
A RunTests script for running many examples and comparing to previous code (in OSParticulas only).
nairn-mpm-fea/Visualization
This folder is the root folder for the platform-independent, Java-based visualization tool.
/src
This folder has the Java source code for the visualization tool called NairnFEAMPMViz.
/build
This folder has a makejar script that can compile the Java source code into a jar file. It also has files for creating a Windows shortcut with a provided icon.
/MatLab_VTK_Viz
Miscellaneous scripts for using MatLab to visualize results include some scripts to read VTK files.
/R_Tools
Miscellaneous scripts for using R to visualize data.
nairn-mpm-fea/Documentation
This folder has all minimal documentation of the current version and links to tech note pdf files. Open the index.html file to view the documentation in any browser.
/TechNotes
This folder has a selection of tech notes as pdf files. In OSParticulas, this folder has LaTeX source files and image files for editing the notes.
