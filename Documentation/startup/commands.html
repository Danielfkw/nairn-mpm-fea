NairnMPM and NairnFEA Input Command Files

The NairnMPM and NairnFEA calculations are controlled by XML commands file. Although you can write an edit raw XML files it is usually preferrable to use the visual front end application that supports a scripting language. In brief, you write a script and then have those applications prepare an XML file. The applications can run and visualize calculations or you can save the XML command file itself and copy to a server for running remote calculations.

Input Command File Helps

Complete help on both scripting commands and XML commands is available on line in the OSUPDocs wiki:

  1. NairnMPM Input Command Files
  2. NairnFEA Input Command Files

The above documentation is only available on line. For times you cannot access the internet, the visual from end Java app called NairnFEAMPMViz has help that briefly describes all scripting commands (the NairnFEAMPMViz.jar application file is included in the project check out). Similarly, the Mac visual front end call NairnFEAMPM has a help window with documentation of all scripting commands.
